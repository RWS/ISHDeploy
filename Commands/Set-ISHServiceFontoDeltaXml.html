Set-ISHServiceFontoDeltaXml

SYNOPSIS

Sets target port for Fonto Delta Xml. This cmdlet is only supported on version 14.0.2.

SYNTAX

Service port

Set-ISHServiceFontoDeltaXml -ServicePort <Int32> [-ISHDeployment <ISHDeployment>]

Configuration

Set-ISHServiceFontoDeltaXml -ServicePort <Int32> -Xmx <String> [-ISHDeployment <ISHDeployment>]

Heap Size

Set-ISHServiceFontoDeltaXml -Xmx <String> [-ISHDeployment <ISHDeployment>]

DESCRIPTION

The Set-ISHServiceFontoDeltaXml cmdlet sets target port for Fonto Delta Xml.

EXAMPLES

---------- EXAMPLE 1 ----------

PS C:\>Set-ISHServiceFontoDeltaXml -ISHDeployment $deployment -ServicePort 8010

This command sets the Fonto Delta Xml target port. Parameter $deployment is a deployment name or an instance of the Content Manager deployment retrieved from Get-ISHDeployment cmdlet.

---------- EXAMPLE 2 ----------

PS C:\>Set-ISHServiceFontoDeltaXml -ISHDeployment $deployment -Xmx 1024k

This command sets maximum of heap size for internal java process of the Fonto Delta Xml. Parameter $deployment is a deployment name or an instance of the Content Manager deployment retrieved from Get-ISHDeployment cmdlet.

---------- EXAMPLE 3 ----------

PS C:\>Set-ISHServiceFontoDeltaXml -ISHDeployment $deployment -Xmx 1024k -ServicePort 8010

This command sets maximum of heap size for internal java process and target port of the Fonto Delta Xml. Parameter $deployment is a deployment name or an instance of the Content Manager deployment retrieved from Get-ISHDeployment cmdlet.
