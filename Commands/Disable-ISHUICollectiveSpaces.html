Disable-ISHUICollectiveSpaces

SYNOPSIS

Disables Collective Spaces for Content Manager deployment.

SYNTAX

DraftSpace

Disable-ISHUICollectiveSpaces [-DraftSpace] [-ISHDeployment <ISHDeployment>]

DraftSpaceAndReviewSpace

Disable-ISHUICollectiveSpaces [-DraftSpace] [-ReviewSpace] [-ISHDeployment <ISHDeployment>]

DraftSpaceAndDocumentHistoryForDraftSpace

Disable-ISHUICollectiveSpaces [-DraftSpace] [-DocumentHistoryForDraftSpace] [-ISHDeployment <ISHDeployment>]

All

Disable-ISHUICollectiveSpaces [-DraftSpace] [-ReviewSpace] [-DocumentHistoryForDraftSpace]
 [-ISHDeployment <ISHDeployment>]

ReviewSpace

Disable-ISHUICollectiveSpaces [-ReviewSpace] [-ISHDeployment <ISHDeployment>]

ReviewSpaceAndDocumentHistoryForDraftSpace

Disable-ISHUICollectiveSpaces [-ReviewSpace] [-DocumentHistoryForDraftSpace] [-ISHDeployment <ISHDeployment>]

DocumentHistoryForDraftSpace

Disable-ISHUICollectiveSpaces [-DocumentHistoryForDraftSpace] [-ISHDeployment <ISHDeployment>]

DESCRIPTION

The Disable-ISHUICollectiveSpaces cmdlet disables Collective Spaces for Content Manager deployment.

EXAMPLES

---------- EXAMPLE 1 ----------

PS C:\>Disable-ISHUICollectiveSpaces -ISHDeployment $deployment -DraftSpace

This command disables Draft Space. Parameter $deployment is a deployment name or an instance of the Content Manager deployment retrieved from Get-ISHDeployment cmdlet.

---------- EXAMPLE 2 ----------

PS C:\>Disable-ISHUICollectiveSpaces -ISHDeployment $deployment -ReviewSpace

This command disables Review Space. Parameter $deployment is a deployment name or an instance of the Content Manager deployment retrieved from Get-ISHDeployment cmdlet.

---------- EXAMPLE 3 ----------

PS C:\>Disable-ISHUICollectiveSpaces -ISHDeployment $deployment -DocumentHistoryForDraftSpace

This command disables Document History. Parameter $deployment is a deployment name or an instance of the Content Manager deployment retrieved from Get-ISHDeployment cmdlet.

---------- EXAMPLE 4 ----------

PS C:\>Disable-ISHUICollectiveSpaces -ISHDeployment $deployment -DraftSpace -ReviewSpace

This command disables Collective Spaces (both Review and Draft features). Parameter $deployment is a deployment name or an instance of the Content Manager deployment retrieved from Get-ISHDeployment cmdlet.

---------- EXAMPLE 5 ----------

PS C:\>Disable-ISHUICollectiveSpaces -ISHDeployment $deployment -DraftSpace -DocumentHistoryForDraftSpace

This command disables Draft Space and Document History. Parameter $deployment is a deployment name or an instance of the Content Manager deployment retrieved from Get-ISHDeployment cmdlet.

---------- EXAMPLE 6 ----------

PS C:\>Disable-ISHUICollectiveSpaces -ISHDeployment $deployment -DraftSpace -ReviewSpace -DocumentHistoryForDraftSpace

This command disables Collective Spaces (both Review and Draft features) and Document History. Parameter $deployment is a deployment name or an instance of the Content Manager deployment retrieved from Get-ISHDeployment cmdlet.
