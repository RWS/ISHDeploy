Move-ISHUIEventMonitorTab

SYNOPSIS

Manipulates with definitions in EventMonitorTab.

SYNTAX

UNNAMED_PARAMETER_SET_1

Move-ISHUIEventMonitorTab -Label <string> [-Last] -ISHDeployment <ISHDeployment>

UNNAMED_PARAMETER_SET_2

Move-ISHUIEventMonitorTab -Label <string> [-First] -ISHDeployment <ISHDeployment>

UNNAMED_PARAMETER_SET_3

Move-ISHUIEventMonitorTab -Label <string> -After <string> -ISHDeployment <ISHDeployment>

DESCRIPTION

The Move-ISHUIEventMonitorTab cmdlet moves Tabs definitions in Content Manager deployment.

EXAMPLES

---------- EXAMPLE 1 ----------

PS C:\>Move-ISHUIEventMonitorTab -ISHDeployment $deployment -Label "Publish" -First

Moves definition of the "Publish" to the top.

---------- EXAMPLE 2 ----------

PS C:\>Move-ISHUIEventMonitorTab -ISHDeployment $deployment -Label "Publish" -Last

Moves definition of the "Publish" to the bottom.

---------- EXAMPLE 3 ----------

PS C:\>Move-ISHUIEventMonitorTab -ISHDeployment $deployment -Label "Translation" -After "Publish"

Moves definition of the "Translation" after "Publish".
