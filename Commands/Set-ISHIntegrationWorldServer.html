Set-ISHIntegrationWorldServer

SYNOPSIS

Sets configuration of WorldServer.

SYNTAX

SOAP

Set-ISHIntegrationWorldServer -Name <String> -Uri <String> [-MaximumJobSize <Int32>]
 [-RetriesOnTimeout <Int32>] [-Timeout <TimeSpan>] [-SOAP]
 -Mappings <ISHLanguageToWorldServerLocaleIdMapping[]> [-ISHDeployment <ISHDeployment>]

PSCredential

Set-ISHIntegrationWorldServer -Name <String> -Uri <String> [-MaximumJobSize <Int32>]
 [-RetriesOnTimeout <Int32>] [-Timeout <TimeSpan>] -Mappings <ISHLanguageToWorldServerLocaleIdMapping[]>
 -Credential <PSCredential> [-UserName <String>] [-UserPassword <String>] [-ISHDeployment <ISHDeployment>]

UserName/UserPassword

Set-ISHIntegrationWorldServer -Name <String> -Uri <String> [-MaximumJobSize <Int32>]
 [-RetriesOnTimeout <Int32>] [-Timeout <TimeSpan>] -Mappings <ISHLanguageToWorldServerLocaleIdMapping[]>
 [-Credential <PSCredential>] -UserName <String> -UserPassword <String> [-ISHDeployment <ISHDeployment>]

DESCRIPTION

The Set-ISHIntegrationWorldServer cmdlet sets configuration of WorldServer.

EXAMPLES

---------- EXAMPLE 1 ----------

PS C:\>Set-ISHIntegrationWorldServer -ISHDeployment $deployment -Name "ws1" -Uri "https:\\ws1.sd.com" -Credential $credential -MaximumJobSize 5242880 -RetriesOnTimeout 3 -Mapping $mapping

This command enables the translation organizer windows service. Parameter -MaximumJobSize is the max value of total size in bytes of uncompressed external job. In the WorldServer configuration section, this parameter has name "externalJobMaxTotalUncompressedSizeBytes". Parameter $deployment is a deployment name or an instance of the Content Manager deployment retrieved from Get-ISHDeployment cmdlet. Parameter $credential is a set of security credentials, such as a user name and a password. Parameter $mapping is a object with pair of properties, where ISHLanguage is InfoShare language identifier retrieved from New-ISHIntegrationWorldServerMapping cmdlet.

---------- EXAMPLE 2 ----------

PS C:\>Set-ISHIntegrationWorldServer -ISHDeployment $deployment -Name "ws1" -Uri "https:\\ws1.sd.com" -UserName $userName -UserPassword $userPassword -MaximumJobSize 5242880 -RetriesOnTimeout 3 -Mapping $mapping

This command enables the translation organizer windows service. Parameter -MaximumJobSize is the max value of total size in bytes of uncompressed external job. In the WorldServer configuration section, this parameter has name "externalJobMaxTotalUncompressedSizeBytes". Parameter $deployment is a deployment name or an instance of the Content Manager deployment retrieved from Get-ISHDeployment cmdlet. Parameter $userName is operation system's user name. Parameter $userPassword is operation system's user password. Parameter $mapping is a object with pair of properties, where ISHLanguage is InfoShare language identifier retrieved from New-ISHIntegrationWorldServerMapping cmdlet.
