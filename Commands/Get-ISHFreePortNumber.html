Get-ISHFreePortNumber

SYNOPSIS

Gets for the free port number in specified range.

SYNTAX

Get-ISHFreePortNumber -StartOfRange <Int32> -EndOfRange <Int32>

DESCRIPTION

This cmdlet gets free port number in specified range.

EXAMPLES

---------- EXAMPLE 1 ----------

PS C:\>Get-ISHFreePortNumber -StartOfRange $startOfRange -EndOfRange $endOfRange
