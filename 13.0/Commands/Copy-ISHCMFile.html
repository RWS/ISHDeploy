Copy-ISHCMFile

SYNOPSIS

Copies several files to Custom or Bin directory and replaces parameters in files on values of the deployment.

SYNTAX

ToCustom

Copy-ISHCMFile -FileName <String[]> [-ToCustom] [-ISHDeployment <ISHDeployment>]

ToBin

Copy-ISHCMFile -FileName <String[]> [-ToBin] [-ISHDeployment <ISHDeployment>]

DESCRIPTION

Will copy files from packages directory to Custom or Bin directory depends on switch and replace the placeholders of files with the corresponding values of the deployment. Replacement of values occurs only in files of certain extensions.

EXAMPLES

---------- EXAMPLE 1 ----------

PS C:\>Copy-ISHCMFile -ISHDeployment $deployment -ToCustom -FileName test.js
